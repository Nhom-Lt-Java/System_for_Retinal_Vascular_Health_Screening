# User Manual

## 1. User (Bệnh nhân)
- **Upload**: User → Upload → chọn 1 hoặc nhiều ảnh → Submit.
- **Kết quả**: User → History → mở từng analysis → xem risk/advice + ảnh explain.
- **Báo cáo**: trong trang Result, tải PDF/CSV.
- **Chat**: vào Chat để nhắn bác sĩ.
- **Thông báo**: trang Notifications.
- **Hồ sơ**: trang Profile để cập nhật thông tin.

## 2. Doctor (Bác sĩ)
- Dashboard/Patients: chỉ thấy bệnh nhân được assign.
- Review: mở Result → xác nhận/chỉnh sửa label, thêm note.
- Trend: Doctor → Trends để xem thống kê theo ngày.

## 3. Clinic
- Bulk Upload: Clinic → Bulk upload nhiều ảnh.
- Doctor manager: tạo/gán bác sĩ.

## 4. Admin
- Users: quản lý user, enable/disable, gán role.
- Pricing: quản lý gói dịch vụ.
- AI Settings: cấu hình threshold.
- Templates: quản lý notification templates.
